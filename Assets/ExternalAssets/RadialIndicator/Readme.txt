	       GUI: Customizable Radial Indicators 
============================================================


	Thank you for your support.
	
	GUi: Radial Indicators is a customizable gui element.



Features
============================================================

	- Customizable Radial Indicator with 2D sprites.
	- Photoshop project for 2D sprites source.
	- Customizable start degrees (0째 : 360째).
	- Customizable range degrees (0째 : 360째).
	- Customizable needle velocity.
	- Selectable rotational sense.



Installation
============================================================

	- Extract the GUI: Customizable Radial Indicators
	  Unity package into your project.
	- Drag one or more example prefab into your scene.
	- Setup each prefab values as you want.
	- That's it!



Customization
============================================================

	You can modify the single sprite and color of an existing
	prefab from the inspector.




FAQ / Troubleshooting
============================================================

	[Q.1] The needle does not move in Editor mode, only in Play mode.

			In Editor mode you can see preview only with Needle
			velocity set to Immediately.
